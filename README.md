# 🔴 Collectif 8 mars 49

Site web du Collectif 8 mars 49 — Angers, Maine-et-Loire.
Édition visuelle via Netlify Visual Editor.

## Déployer

1. Upload tous ces fichiers dans un nouveau repo GitHub
2. Sur netlify.com → Add new site → Import from Git → sélectionne le repo
3. Netlify déploie automatiquement
4. Active le Visual Editor dans Project configuration

## Modifier le site

Tes collègues vont dans Netlify → Visual Editor et modifient tout visuellement :
- Textes, titres, descriptions
- Associations (ajouter, modifier, supprimer)
- Événements
- Couleurs et style
- Formulaire de contact
